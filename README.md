## 描述

[WSBM]() 唯守书签前端项目。

## 安装

```bash
$ npm install
```

## 运行

```bash
# 开始
$ npm run start

# 观察
$ npm run watch
```

## 构建

```bash
$ npm run build
```

## 创建组件

```bash
$ ng g component xxx --skip-tests=true
```
